# AI for Science - OpenApi specification

The AI for Science Catalog API specification is defined using [Azure TypeSpec](https://azure.github.io/typespec-azure/docs/intro) which compiles to [OpenAPI](https://swagger.io/specification/) JSON, also known as _Swagger_. 

Please follow the Science team's Swagger [development process](https://dev.azure.com/msazuredev/AIforSciencePlatform/_git/science-rp-infrastructure?path=/documentation/swagger/api-spec-update-process.md&_a=preview) for updating this API spec.
